Visualização de Dados com Matplotlib e Seaborn

1. Importando bibliotecas:
import matplotlib.pyplot as plt
import seaborn as sns

2. Gráficos básicos com Matplotlib:
# Gráfico de linha
plt.plot(x, y)
plt.title('Título do Gráfico')
plt.xlabel('Eixo X')
plt.ylabel('Eixo Y')
plt.show()

# Gráfico de barras
plt.bar(x, y)
plt.show()

# Gráfico de dispersão
plt.scatter(x, y)
plt.show()

3. Gráficos com Seaborn:
# Histograma
sns.histplot(data=df, x='coluna', bins=30)
plt.show()

# Box plot
sns.boxplot(data=df, x='coluna')
plt.show()

# Violin plot
sns.violinplot(data=df, x='coluna')
plt.show()

4. Personalização:
# Estilo
plt.style.use('seaborn')
sns.set_theme()

# Cores
plt.plot(x, y, color='red', linestyle='--', marker='o')

# Legendas
plt.legend(['Série 1', 'Série 2'])

5. Subplots:
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))
ax1.plot(x1, y1)
ax2.plot(x2, y2)
plt.tight_layout()
plt.show()
